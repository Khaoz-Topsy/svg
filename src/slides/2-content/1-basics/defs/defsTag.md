# The defs tag

- the defs tag is used to store definitions, it's content isn't displayed until referenced later
- It can help you keep your SVG clean and modular

## Top Left

- Symbol allows us to group multiple shapes
- We can reuse the shapes with the **use** tag and the **id**

## The others

- We can define a whole lot of things like gradients, filters, patterns and more
- Again referencing them by **id**
